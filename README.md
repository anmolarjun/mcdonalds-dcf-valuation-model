# McDonald's DCF Valuation Model

## Overview

This repository contains a comprehensive **Discounted Cash Flow (DCF) valuation model** for **McDonald's Corporation (NYSE: MCD)**, developed in Microsoft Excel. The model estimates the intrinsic value of the company by forecasting future **Free Cash Flows to the Firm (FCFF)** and discounting them using the **Weighted Average Cost of Capital (WACC)**.

The project demonstrates practical financial modeling and valuation techniques commonly used in **Investment Banking, Equity Research, Corporate Finance, FP&A, and Private Equity**.

## Project Objectives

* Build an integrated DCF valuation model in Excel.
* Forecast future financial performance using historical financial statements.
* Calculate Enterprise Value and Equity Value.
* Estimate the intrinsic value of McDonald's using the FCFF approach.
* Apply professional financial modeling best practices.

## Model Components

* Historical Income Statement

* Financial Forecasts
* Free Cash Flow to the Firm (FCFF)
* Weighted Average Cost of Capital (WACC)

* Enterprise Value
* Equity Value
* Implied Share Price

## Valuation Methodology

The valuation follows the standard DCF process:

1. Analyze historical financial statements.
2. Forecast operating performance.
3. Calculate Free Cash Flow to the Firm (FCFF).
4. Estimate the Weighted Average Cost of Capital (WACC).
5. Calculate Terminal Value using the Gordon Growth Model.
6. Discount projected cash flows to present value.
7. Determine Enterprise Value and Equity Value.
8. Estimate the intrinsic value per share.
